# Locale

Robertify has **7** different languages for you to choose from. These include
`Englisgh (UK)`, `Spanish`, `French`, `German`, `Dutch`, `Portuguese` and `Russian`.


You can easily switch your locale by running the `language` command.

<Callout type='warning'>
    This is an **admin-only** command.
</Callout>
